定时规则1：第一组定时假如18:20~18~30,第二组定时18：25~18:35,
则第一组定时正常开启和关闭，第二组定时不会在设置的开启时间开启，
但会在第一组关闭时间开启，直到第二组关闭时间，此时第二组就会关闭！

定时规则2：第一组定时假如18:20~18~40,第二组定时18：30~18:35,
则第一组定时正常开启和关闭，第二组定时不会则不会开启（设定的时间段）。

定时规则3：第二组定时比第一组提前，此时到达设定时间，两组都不会开启！

输出脚4脚和5脚，即D1和D2;

// 定义输出引脚
const int outputPinForGroup1 = 4;
const int outputPinForGroup2 = 5;

// 定义 OLED 的 I2C 引脚
const int sclPin = 1;
const int sdaPin = 3;

经测试只能4和5能用，其它脚在上电的一瞬间有高电平输出，ESP8266不能开启，
或屏幕不显示等问题！
